# Raytracing

A simple real-time ray tracer with a controllable camera.

![demo image](demo_light.png)

## Getting Started

### Dependencies
* [SDL2](https://www.libsdl.org/) - window creation
* [glm](https://github.com/g-truc/glm) - math library

### Installing
```
>>> make build
```

### Executing program
```
>>> ./raytracer
```
